This is a game which involves 2 players rolling 2 die: 
1. You can set a final score, the first player to reach that score wins. 
2. The score increases by rolling the die and their combined value is calculated.
3. The score can be banked by pressing the hold button, it also changes the active player.
4. If a player rolls a 1, they loose there unheld score.
5. If a final score is not entered, 100 is taken as the default value.
